AI Job Auto Apply System

An AI-powered automated job application system built with Python, Selenium, and AWS.

Features

  • Resume parsing and skill extraction
  • Job scraping and AI-based matching
  • Automated form filling using Selenium
  • Cloud deployment on AWS EC2

Tech Stack

Python, Selenium, BeautifulSoup, AWS EC2, OpenAI API (optional)
